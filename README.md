## 기술 스택

- **단일 파일** — `index.html` (프레임워크·빌드 없음)
- **PWA** — `manifest.json` + Service Worker → 오프라인 동작
- **배포** — Vercel (GitHub `main` 푸시 시 자동 배포)
- **테마** — CSS 변수 5종 (green / blue / rose / earth / slate)
